Project Description

Choosing the right college is an arduous process. There are so many factors to consider, especially academically, financially, and geographically. Instead of struggling over endless lists of prospective schools that meet your preferences, budget, and scores, what if you could get a list of colleges tailored perfectly to your needs and potential? Our Python program, College Matcher, will take your SAT score, a U.S. state, and your budget as inputs and based on these factors, will return a college (in the top 650 ranked schools in the U.S.) in your chosen state that you have a good shot of getting into. The program will provide the user with not only the name of the suggested university, but also any other relative information retrieved from our data set (city, size, acceptance rate, and school website).

Many students prefer to stay in their home states or visit a certain state for college, and interacting with College Matcher would be an easy way for them to get a list of colleges that work with both their geographical and financial needs and test scores. We want to make this program because as we enter our junior year, we are overwhelmed by the sheer number of colleges out there. We think it would be helpful to create a program that narrows down that list of choices based on the aforementioned criteria.

The program will ask the user for 3 inputs: a U.S. state, an SAT score, and a budget. The program will then use these criteria to find the highest-ranked university that meets the user’s criteria. For this program, we will be using a data set that ranks the top 650 universities in the U.S., which includes relevant information (scores, tuition, location). We will use the pandas package to create a dataframe from the .csv document. Pandas is a widely used package in Python that is used in data analysis. It allows the user to create dataframes which in turn enable the programmer to perform data analysis in a quick and efficient manner. We will use the user input to slice and categorize the data.

Instructions

This program is very user-friendly. The program prompts you to enter each of the following inputs: US state(s) two-letter abbreviations separated by a space, your SAT score, and your budget. It will then return a school and some helpful background information about the college, such as the city in which it is located, the student population, the average alumni salary, and a link to its website. If there are no matches for your entered criteria, the program will return an Error message.

How College Matcher Works

The college information (includes everything from the SAT score range to the acceptance rate of the top 650 schools in the United States) is stored in a .csv file called “ForbesAmericasTopColleges2019.” However, not all the information was useful for our program, so we used the “.drop()” pandas feature to modify the file. We removed the ‘Undergraduate Population', 'Net Price', 'Average Grant Aid', 'ACT Lower', and 'ACT Upper' data columns. The program sends a welcome message to the user, prompting them to get 3 pieces of information ready: their SAT score, their budget, and which state(s) in which they would like to attend college. We then utilized “input” to incorporate an interactive element. The program asks the user for their 3 pieces of information. We modified our first draft of the program by adding a feature that allows the user to enter more than one state as an input. We used “.split()” to achieve this, and for that reason, the states must be separated by a space when entered into the program. We put the different states in a new list called “states.” Then, we used pandas’ data slicing capabilities to narrow down the table only the rows that met the criteria entered by the user. From there, we selected the first row in the new table, or the highest ranked college that meets the criteria. Finally, the program runs a script that gives the user their top college and some information about it.
